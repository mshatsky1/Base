# Compatibility

## Requirements

This repository has minimal requirements:

- Git 2.0+ for version control
- A text editor for markdown files
- Optional: markdownlint for linting

## Platform Support

- **macOS**: Fully supported
- **Linux**: Fully supported
- **Windows**: Fully supported (with Git for Windows)

## Tools

- Any markdown viewer or editor
- GitHub web interface
- Local git clients

## Future

Compatibility requirements will be updated as new tools are added.
