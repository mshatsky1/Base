# Support

## Getting Help

This repository is a learning and experimentation space. For questions:

1. Check the [FAQ](docs/faq.md) for common answers.
2. Review existing [documentation](docs/).
3. Open an issue for bugs or feature requests.

## Community Guidelines

- Be respectful and constructive.
- Search existing issues before creating new ones.
- Provide clear descriptions when reporting problems.

## Maintenance

This project is maintained on a best-effort basis. Response times may vary.
